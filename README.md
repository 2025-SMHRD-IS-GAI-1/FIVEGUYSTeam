# FIVE GUYS - 다국어 메뉴판 OCR·번역·설명 서비스

업로드한 **메뉴판 이미지를 OCR로 인식**하고,  
선택한 언어로 **메뉴 이름과 설명을 번역**해 주는 웹 서비스입니다.  
해외 식당에서 메뉴를 이해하기 어려운 사용자를 위해 번역 + 메뉴 설명 + 부가 정보(알레르기 등)를 한 번에 제공합니다.

---

##  주요 기능

- **이미지 업로드 & 미리보기**
  - 사용자가 메뉴판 이미지를 업로드
  - 업로드한 이미지를 화면에서 확인

- **OCR 문자 인식**
  - 업로드한 이미지에서 메뉴 텍스트를 추출
  - 추출된 영역별 좌표 정보를 함께 저장 (X1, Y1, X2, Y2)

- **다국어 번역 & 메뉴 설명**
  - 번역 대상 언어 선택 (예: 한국어, 영어, 일본어, 중국어 등)
  - 메뉴 이름과 설명을 번역해서 한눈에 표시
  - (선택) 알레르기/성분 등 부가 설명 제공

- **계정 및 마이페이지**
  - 회원가입 / 로그인 / 로그아웃
  - 마이페이지에서 프로필, 비밀번호 변경, 즐겨찾기 이미지 관리
  - 비회원(게스트 모드)은 번역만 가능, **저장/즐겨찾기/이력 조회는 불가**

- **이미지/번역 기록 관리**
  - 회원별 업로드 이미지 이력 조회
  - 이미지별 번역 결과(메뉴명, 번역문, 설명) 조회

- **관리자 기능**
  - 회원 목록 조회, 권한 관리(일반 / 관리자)
  - 부적절한 이미지나 텍스트 비공개 처리

##  주요 테이블 구조 (요약)

※ 자세한 내용은 `/docs` 폴더의 **테이블 명세서 / DB 요구사항 분석서** 등을 참고하도록 안내해도 좋음.

### 1) 회원 테이블 `T_MEMBER`

- `ID` : 회원 아이디 (PK)
- `PW` : 비밀번호 (해시 저장)
- `NAME` : 이름
- `EMAIL` : 이메일
- `ADMIN_YN` : 관리자 여부 (`M`: 일반, `A`: 관리자 등)
- `JOIN_DT` : 가입 일자

### 2) 이미지 테이블 `T_IMAGE`

- `IMG_ID` : 이미지 ID (PK, `SYS_GUID()`)
- `ID` : 회원 아이디 (FK → `T_MEMBER.ID`)
- `IMG_NAME` : 업로드한 이미지 이름
- `IMG_FILE` : 이미지 파일 (BLOB)
- `RES_NAME` : 식당 상호명
- `ADDR` : 주소
- `LAT`, `LON` : 위도/경도
- `RATINGS` : 평점
- `IMG_CHECK` : 체크 여부
- `UPLOAD_DT` : 업로드 일자

### 3) 번역 테이블 `T_TRANSLATION`

- `TRANS_ID` : 번역 ID (PK, `SYS_GUID()`)
- `IMG_ID` : 이미지 ID (FK → `T_IMAGE.IMG_ID`)
- `X1, Y1, X2, Y2` : 메뉴 텍스트 영역 좌표
- `MENU_NAME` : 메뉴명
- `TRANS_TEXT` : 번역된 메뉴명/텍스트
- `MENU_DESC` : 메뉴 설명
